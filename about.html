Purpose

The Digital Governance Ranking system evaluates companies based on their implementation of digital governance practices. This assessment helps identify organizations that demonstrate strong commitment to responsible digital transformation and ethical technology use.

Methodology

Each indicator is assessed based on:

  • Documented policies and procedures
  • Implementation evidence
  • Employee feedback and training
  • External verification where applicable

For example, a company might score:

  • Core Indicator 1: 2/3 (Strong)
  • Core Indicator 2: 1/3 (Basic)
  • Specialized Indicator 1: 3/3 (Advanced)

This would result in an average score of 2.0, which would be scaled to 6.67/10.

Scoring System

Each indicator is scored on a scale of 0-3:

  • 0 - Not Implemented
  • 1 - Partially Implemented
  • 2 - Mostly Implemented
  • 3 - Fully Implemented
  • N/A - Not Applicable

Overall Score Calculation

The overall score is calculated using the following steps:

  1. Filter Applicable Indicators: Only indicators with actual scores (excluding N/A) are considered in the calculation.
  2. Calculate Average: The average of all applicable indicator scores is computed.
  3. Scale to 0-10: The average (which is on a 0-3 scale) is converted to a 0-10 scale using the formula: (average / 3) × 10
  4. Visual Representation: The final score is displayed as a 10-segment bar, where each filled segment represents one point.

For example:

  • If a company has scores of 2, 3, and 1 on three applicable indicators:
  • Average = (2 + 3 + 1) ÷ 3 = 2
  • Scaled Score = (2 ÷ 3) × 10 ≈ 6.67
  • This would be displayed as 6.7/10 with 7 filled segments in the score bar
